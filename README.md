# Square Sweep Site

Public GitHub Pages site for Square Sweep support and App Store policy URLs.

## URLs

- Site: `https://randyventures.github.io/Square-Sweep-Site/`
- app-ads.txt: `https://randyventures.github.io/Square-Sweep-Site/app-ads.txt`
- Privacy Policy: `https://randyventures.github.io/Square-Sweep-Site/privacy/`
- Support: `https://randyventures.github.io/Square-Sweep-Site/support/`

## Structure

- `site/index.html` - public landing page
- `site/app-ads.txt` - AdMob app-ads.txt seller declaration
- `site/privacy/index.html` - privacy policy
- `site/support/index.html` - support and FAQ
- `.github/workflows/pages.yml` - GitHub Pages deployment

## Deployment

Deploys from the `main` branch through GitHub Actions Pages.
